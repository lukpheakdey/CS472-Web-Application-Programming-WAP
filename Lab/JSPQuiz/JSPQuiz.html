JSP Introduction Quiz

Question 1: What is the main value of Java Server Pages?

Answer: JSP separates display from processing, i.e., separate html from java. JSP is html with a little java mixed in.

Question 2: How are JSP pages related to servlets?

Answer: Server converts the JSP into a servlet class that implements the javax.servlet.jsp.JspPage interface. The JSP file is compiled only when the page is first requested, or when the JSP file has been updated and has a more recent timestamp.

Question 3: How are JSP pages related to HTML?

Answer: JSP already contains HTML which means JSP allows us to include HTML.

Question 4: Give an example of a JSP scriptlet and show how it will look in the JSP servlet .

Answer: <% int value = 10; %>
It will be considered as local variable in the service method.

Question 5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

Anser: <%! int value = 10; %>
It will be considered as globel variable in the class.

Question 6. Give an example of a JSP comment and show how it will look in the JSP servlet.

Answer : <%-- write Comment here --%>

Question 7. Give an example of a JSP expression and show how it will look in the JSP servlet.

Answer : <%= count++ %>

Question 8. Give an example of a JSP directive and show how it will look in the JSP servlet.

Answer : <%@page import="java.util.*" %>

Question 9. Explain how an EL expression is evaluated.

Answer : The JSP implementation evaluates the EL expression, converts it, and passes the returned value to the tag handler.
${something} - container evaluates this as follows
It will check in the all the scope respectively page, request, session, application. If it does not found it will be ignored.

Question 10. Explain how servlet attributes are involved in EL expressions

Answer: There is four types of scopes. EL expressions are invloed in four scope.
1- Page Scope.
2- Request Scope.
3- Session Scope.
4- Application Context Scope.

Question 11: Explain how servlets and JSPs use request dispatch to interact.

Answer: We use dispatch to send data as request and response to other jsp.
